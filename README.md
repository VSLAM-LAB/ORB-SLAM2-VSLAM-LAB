# ORB-SLAM2

This repository is a fork of [**ORB-SLAM2**](https://github.com/raulmur/ORB_SLAM2), incorporating updates to utilize most recent libraries.

This modified version serves as a baseline for Visual Simultaneous Localization and Mapping (VSLAM) experiments within the  [**VSLAM-LAB**](https://github.com/VSLAM-LAB/VSLAM-LAB) environment.
